# Contao 5 Theme Tapestry
Website-Template für Contao 5.7 via Twig Layout mit Slots

Wer nur das Theme haben möchte, ohne die Datenbank Daten (Seiten/Artikel), es gibt hier auch eine cto Datei für den Import im Backend -> Layout -> Themes<br>
[Download tapestry_for_contao_5.cto](https://github.com/BugBuster1701/Contao_5_Theme_Tapestry/raw/refs/heads/main/files/tapestry_for_contao_5.cto)

Das Original Design stammt von HTML5Templates.com, wurde damals zu einem Contao 3.5 Theme umgebaut und ist nun für Contao 5.7 angepasst.
